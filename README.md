# Bootstrap Ruby on rails & Angular 4

Starter app for ruby on rails and angular 4

### Software Requirements
  * NodeJs v6.10.3
  * Ruby on Rails 5.1.3
  * angular-cli 1.3.1

### Getting Started
    git clone git@github.com:alenteria/rails-angular4.git
    cd rails-angular4
    bin/bundle
    bin/ng-serve
    bin/ng-build
